# github-api-exercise


#### Projemin amacı

Kullanıcı adı girilen Github kullanıcısının kullandığı dilleri ve oranlarını listelemektir.

### Gerekli paketlerin yüklenmesi için:

```
npm install
```

### Projeyi çalıştırmak için:

```
npm run serve
```

### Build almak için:

```
npm run build
```

### Kullanılan Paketler:

```
    "axios": "^0.21.4",
    "core-js": "^3.6.5",
    "vue": "^3.0.0"
```
